# Context

## Glossary

### 文化展览式互动体验

一种面向普通公众和学生的女书网页体验定位。页面第一眼应像精致的数字展厅或文化故事体验，而不是研究问卷工具；研究流程、逐句 TTS、反馈和分享仍保留，但应融入故事阅读和声音体验的叙事中。

### 纸本档案 + 声音仪式感

一种女书文化体验的视觉气质。用纸张纹理、竖向批注、墨色/朱砂点缀、女书字符大字形、句子高亮和声音波形暗示“阅读、聆听、记录”的仪式感，避免泛民俗旅游风或通用科技感。

### 线性研究流程 + 故事体验优先

一种体验结构。用户仍按“开场、阅读前记录、女书故事逐句点读、体验后反馈、保存/分享/了解更多”的顺序完成体验，但页面的第一印象应优先呈现故事、文化和声音，而不是问卷或研究后台。

### 真实女书字形素材

女书字符本身应使用真实来源的字形或图片，不能用 AI 生成的伪女书字。纸张、档案桌面、墨迹纹理、声音波形、抽象展陈和其他文化氛围图可以用生成图，但不得伪装成真实历史照片、真实文物或真实馆藏。

### 字体渲染的女书字形视觉

在没有明确授权的女书图片资产前，优先使用项目内真实女书字体渲染 Unicode 女书字符，并通过纸张、印痕、放大裁切、竖排批注等方式形成图片化视觉。后续如获得真实授权图片，可替换为图片资产。
